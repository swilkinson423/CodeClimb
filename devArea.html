Web Development

Web Development is a term which can be used when refering to the design and construction of websites, plugins, or web-based mobile sites, as well as server-based web applications, databases, or behind-the-scenes web functionality.

Mobile Development

Developing for mobile devices can include creating games and apps for any mobile platform, however the main areas of focus are Android and iOS phones and tablets.

Software Development

Computer Software Development refers to the development of software for Personal Computers. This can include anything from internet browsers, calculators, and word processing software to calendars, email programs, and much, much more.

Data Analysis

Data analysis is an area of programming focusing on using computers to dig through large amounts of data in order to find patters and outliers, or to organize the data in more manageable forms.

Game Development

Game development can take many forms, from creating very simple mobile or web-based 2D games, all the way to developing AAA games for home consoles and gaming computers.

Other Programming Areas

While much of programming can be categorized in the previous areas, programming can be useful for so many other things. From developing self-driving cars, wearables and VR/AR devices, connected objects such as smart watches, or just for creating simple scripts for making repetitive tasks more manageable.
